# # Chidubem Esther Onu - Personal Website

A sleek, modern personal website showcasing the professional journey of Chidubem Esther Onu, a Registered Nurse transitioning from bedside care to telehealth and virtual care.

## Features

- **Responsive Design**: Fully responsive layout using Tailwind CSS
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Sections Include**:
  - Hero section with profile photo
  - About Me with key strengths and skills
  - Professional Experience timeline
  - Education background
  - Scrubs2Screens posts showcase (9 posts documenting the transition journey)
  - Contact information

## Scrubs2Screens Journey

The website prominently features content from the Scrubs2Screens community, documenting the transition from traditional nursing to digital health, including topics on:
- Telenursing and virtual care
- Digital health in Africa
- Healthcare collaboration
- Nurse burnout and sustainability
- Remote patient monitoring

## Technologies Used

- HTML5
- Tailwind CSS (via CDN)
- Font Awesome Icons
- Google Fonts (Inter)
- Vanilla JavaScript for interactivity
